# 🌱 The Root

**나는 미국 간호사 여정 어디까지 왔을까?**

The Root는  
한국 간호사들이 미국 간호사(RN)가 되기까지의 과정을  
단계별로 이해할 수 있도록 만든 인터랙티브 가이드입니다.

👉 5가지 질문만으로  
지금 나의 위치와 다음 단계를 확인할 수 있어요.

---

## ✨ Live Demo

👉 https://your-link.vercel.app  
(여기에 너 Vercel 링크 넣기)

---

## 🧭 What this project does

- 미국 간호사 준비 과정을 단계별로 구조화
- NCLEX, 임상경력, 비자스크린 등 핵심 흐름 정리
- 사용자가 자신의 현재 위치를 파악할 수 있는 퀴즈 기반 UX
- 다음 단계까지 명확하게 제시

---

## 👩🏻‍⚕️ Who this is for

- 미국 간호사를 준비하는 한국 간호사
- NCLEX 준비 중인 분들
- 비자스크린 / 취업 과정이 헷갈리는 분들

---

## 💡 Why I built this

미국 간호사 준비 과정에서  
가장 어려웠던 건 "정보"가 아니라

👉 **전체 흐름이 보이지 않는 것**이었어요.

각각의 단계는 알겠는데  
순서가 연결되지 않아서  
계속 헤매는 느낌이 들었거든요.

그래서  
👉 제가 직접 경험한 기준으로  
하나의 흐름으로 정리해봤습니다.
